IronForge Gym Management System
Project Overview
IronForge Gym Management System is a simple web application developed using Flask, HTML, and CSS. The project is designed to provide users with information about the gym, its facilities, and membership plans through a clean and responsive website.
The website contains multiple pages including Home, About Us, and Membership. Flask is used to manage routing and template rendering, while HTML and CSS are used for the frontend design.

Purpose of the Project
The purpose of this project is to demonstrate the basics of web development using Flask. It helps beginners understand:
Flask routing
Template rendering
Static file management
Basic frontend design with HTML and CSS
Website structure and organization

Technologies Used
Python
Flask
HTML5
CSS3

How to Run the Project Locally
1. Extract the ZIP File
Extract the project folder on your computer.
2. Open Terminal or Command Prompt
Navigate to the project directory:
cd gym_app
3. Install Flask
pip install flask
4. Run the Application
python app.py
5. Open in Browser
Visit the following link in your browser:
http://127.0.0.1:5000
